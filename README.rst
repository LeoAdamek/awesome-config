--------------
 Installation 
--------------

Install is simple!

.. code::
    rm -rf ~/.config/awesome
    git clone https://github.com/LeoAdamek/awesome-config ~/.config/awesome
    cd ~/.config/awesome
    git submodule init && git submodule update
